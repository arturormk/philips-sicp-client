# Code of Conduct

This project expects professional, constructive participation.

## Expected Behavior

- Be respectful and direct.
- Keep discussion focused on the technical issue or contribution.
- Assume protocol behavior can vary by display model and firmware.
- Provide evidence when reporting command behavior, such as packet logs,
  display model, firmware version, and exact CLI commands.

## Unacceptable Behavior

- Harassment, threats, personal attacks, or discriminatory language.
- Publishing private network details, credentials, or other sensitive data.
- Repeated off-topic comments that make project maintenance harder.

## Enforcement

Project maintainers may edit, hide, or remove comments and issues that violate
this code of conduct. Maintainers may also block users who repeatedly disregard
these expectations.

If you need to report a conduct issue, contact the project maintainer through
the GitHub repository once it is published.
